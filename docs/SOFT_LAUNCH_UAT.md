# Soft Launch UAT Checklist

## Suggested launch group

- 1 admin
- 2 teachers
- 10-20 students
- 2-3 batches

## Student checks

- sign up with email verification
- sign in with valid credentials
- password reset works
- enrolled batches are visible
- lectures and notes open correctly
- tests can be attempted and reviewed
- doubts can be posted and replied to
- live classes appear correctly
- notifications list loads
- downloads work, and invalid downloads are handled safely

## Teacher checks

- login works on web
- own courses are visible
- can create/edit content
- can create tests
- can manage only own doubts/live classes
- analytics pages load correctly

## Admin checks

- login works on web
- can manage students and teachers
- can manage courses and batches
- can enroll and remove students
- can create and delete announcements
- analytics pages load correctly
- payment module is not exposed in this release

## Signoff rule

Soft launch passes only if all three roles complete their critical flows without blocker bugs.
